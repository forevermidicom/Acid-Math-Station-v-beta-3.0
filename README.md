# Acid-Math-Station-v-beta-3.0
Acid Math Station is a web-based video synthesizer built on real-time GLSL shader programming, where audio, visuals, and code operate as a generative system. Drawing from the NFT Acid Math series concept and grounded in the idea of “chaos out of order”, it uses randomized mathematical structures, equations, and feedback systems to turn audio and video inputs into unpredictable visual forms. Order becomes a catalyst rather than a constraint: precise logic gives rise to instability, recursion, and transformation. The result is a stream of evolving compositions. The system allows live capture of both audio and video streams, with the ability to record and export high-resolution compositions up to 4K.
